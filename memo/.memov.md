# Normal Mode
- a: Insert after
- \*/#: Next/prev occurence of word
- s: Delete char and insert
- S: Delete line and insert
- C: Delete rest line and insert

# Insert Mode
- C-w: Delete word
- C-t: Indent line

# Tabs
- (num) g t/T: Go to tab
- C-w c: Close window

# Select Mode
- gh: Enter select mode
